# Rally ACU Website

Welcome to the official website repository for **Rally ACU**, a premier student-led society at Ahram Canadian University. Our mission is to develop future leaders, innovators, and entrepreneurs through workshops, competitions, sports leagues, and community initiatives.

---

## ⚡ Features

- **Ambient Background**: Interactive canvas-based particle system with mouse-repulsion physics.
- **Dynamic Theme Manager**: Fully responsive dark and light modes with system preference syncing.
- **Scroll Improvements**: Smooth scrolling, customized scroll progress bar, and a back-to-top helper with circular progress indicator.
- **Interactive Team Directory**: Search and filter team members dynamically by name or committee (Marketing, Entrepreneurs, Operations, Business Development, Talent Management).
- **Interactive FAQ Accordion**: Interactive questions and answers panel with CSS transition animations.
- **Image & Video Galleries**: Lightbox support for image/video viewing on the Seasons page.
